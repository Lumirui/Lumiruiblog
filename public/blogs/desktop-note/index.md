偶然的机会，开始做 Python CV 匹配，做桌面应用。

## 随想

Python 为什么和 AI 这么源远，因为免费和学术习惯。大家都用 python，积累都在 python 上。

程序员都能分析出，Python 不是最优解，肯定有更好的搭配。但普通人来说，普通开发来说，就是 Python。

就是巧合。

我也巧合，以前开发一个应用，自然习惯用 Electron，现在更爱用 Python + Frontend 一套。或者尝试用 Pake。新的经验对以后的开发选择有新的思路。

## 收获

我发现，我无法像知识博主、游戏博主、vlog 博主去输出那些吸引人，有用、有趣的视频，我上班是没有那么充分的时间和精力。

我能做的产出，似乎就是擅长的办公式应用开发，顺手开发一个 App。有明确的架构、规划，快速产出或许有用的东西，AI 辅助下，有更宽泛的领域，更快的产出，更少的时间占用。

重要的是，AI 辅助下，可以进行碎片式开发。这很革命式，以前开发一款东西，需要沉浸式，冥想者一般，半个小时进入状态，2小时高效思考，然后休息，停断。

现在，AI 让你碎片的去做，写好 Todo，做好限定架构，碎片开发模块的文件。像个工人。

## 有限的收获

问题是，就算这种开发，产出一个 App，收获很小。远远比不过上班，并且工作后的精力本身也有限。

累久之后，会时不时拷问，值不值得？

## 若以前

若以前，学习、工作、积累、拓宽、尝试、研究、开源，这些字眼，传递自己一个正向进步的感受，知道自己在进步，知道这样下去，工作会如意，以后下一份工作也应该有着落。

## 这样以后

我以后，能靠视频博主赚钱吗？视频收入，在 B站，是很低的，不同赛道，收入都不同，但大抵不多。在年轻的时候，主要精力放在工作上班上，是肯定的。

年纪大了，工作或许不好找了，可能是“自由”的选择？

我喜欢，自己开发的东西，有人用。但也需顾好自己的材米油盐。

## 停一下

今天，零零碎碎的记录。我知道，不会想出什么样的结果。

但我突然想到，无论什么，我都先停一下。停一下，先好好工作，好好休息。

后面，我肯定会做出选择，但不必是现在。
